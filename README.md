## Development
### 1. Initial Setup
* Clone the project repo
    ```
    git clone https://github.com/usmanasif/atm-api.git
    ```
* Get into the project directory
    ```
    cd atm-api
    ```
* Install gems and their dependencies
    ```
    bundle install
    ```
### 2. Start App Server
```
rails server
```
### 3. Running Tests
```
bundle exec rspec
```
